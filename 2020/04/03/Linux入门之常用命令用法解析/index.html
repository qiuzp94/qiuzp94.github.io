Linux入门之常用命令用法解析

发表于2020-04-03更新于2020-04-03

常用命令

ls

意思：查看路径（list-罗列）
用法：
ls [参数][文件名]

-a：显示所有文件与目录
-l：除文件名称外，亦将文件型态、权限、拥有者、文件大小等资讯详细列出

例子：

  • ls -a 路径
  • ls -l 路径
  • ls -al 路径

cat

意思：显示或把多个文本文件连接起来–可以预览文件
用法：
cat [参数][文件名]
例子：
cat 文件

mv

意思：移动文件（move-移动）
用法：
mv [源文件][目标文件]
例子：
mv 文件名 文件名 — 将源文件改名为目标文件名
mv 文件名 目录名 — 将文件移动到目标目录

touch

意思：用于修改文件或者目录的时间属性，包括存取时间和更改时间。若文件不存在，系统会建立一个新的文件。
用法：
touch [参数][文件]

-a 改变档案的读取时间记录。
-version 列出版本讯息。

例子：
touch 1.txt

tree

意思：查看目录结构
用法：tree [参数][文件]
例子：
tree 文件名

rm

意思：删除文件（remove）
用法：rm [参数][文件]
-f 强制删除
-r 递归删除
例子：

  • 删除文件 rm 文件路径
  • 强制删除文件 rm -f 文件路径
  • 删除目录 rm -r 目录路径
  • 强制删除目录 rm -rf 目录路径

pwd

意思：显示当前目录
用法：pwd
例子：pwd

mkdir

意思：创建目录
用法：mkdir [参数][目录名]
例子：

  • 创建目录 mdir 目录名
  • 创建目录 mkdir -p 目录路径

cd

意思：进入目录 (change directory)
用法：cd [目录名]
例子：
cd 目录名

echo

意思：在显示器上显示一段文字，一般起到一个提示的作用。
用法：echo [ -n ] 字符串
例子：

  • 创建文件 echo '1' > 文件路径
  • 强制创建文件 echo '1' >! 文件路径
  • 追加文件内容 echo '1' >> 文件路径

ln

意思：建立软链接
用法：ln [参数] [真实文件][链接]
例子：
ln -s 真实文件 链接

curl

意思：下载文件
用法：curl [参数] [网站] [存储目录]
例子：
curl -L https://www.baidu.com > baidu.html

wget

意思：wget是linux上的命令行的下载工具
用法：wget [参数列表] URL
例子：
拷贝网页wget -p -H -e robots=off https://www.baidu.com (Windows 不支持 wget)

find

意思：用来在指定目录下查找文件
用法：find [起始目录] 寻找条件 操作
例子：ind /-name yum.conf

其他命令

  • 磁盘占用 df -kh
  • 当前目录大小 du -sh .
  • 各文件大小 du -h
