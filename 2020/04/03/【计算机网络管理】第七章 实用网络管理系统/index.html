本章重难点

本章一般会出选择题、填空题、分值在5分左右。

本章的重难点和常考知识点

  • 网络管理系统软件（常考选择题、填空题）
  • Cacti图形化网络管理系统(常考选择题、填空题)
  • 使用Cacti管理网络管（常考选择题、填空题）

一、网络管理系统概述

网络管理系统是实现网络功能、保障网络正常运行的软硬件组成的综合系统。

二、网络管理系统软件（平台）

2.1 CiscoWork

2.2 华为Quidview

华为公司针对IP网络开发的适合于各种网络管理的网络管理软件，是iMessager系统网络管理产品之。

特点

  • 图形化管理界面
  • 操作简单
  • 运行环境与平台无关（Java开发）
  • 中英文显示

2.3 其他网络管理系统

  • HP OpenView
  • IBM Tivoli NetView(Unix)
  • SNMPc(分布式、大型)
  • Cabletron Sepctrum(智能)
  • SolarWinds Orion
  • StarView（Windows）

三、Cacti图形化网络管理系统

3.1 Cacti 系统特点

  • 免费开源
  • 跨平台
  • 界面友好
  • 多种数据采集方式
  • 丰富的模板
  • 可扩展性

3.2 运行条件

  • Apache
  • PHP
  • MySQL
  • SNMP

3.3 组成部分

  • 拓扑管理器
    • 需安装Weathermap插件
  • 事件管理器
    • 日志查看工具查看系统报警日志
    • Syslog插件扩展系统日志功能
    • Settings插件实现邮件报警
    • Thold插件设置设备报警阈值
  • 性能管理器
    • 可以同时监视多个网络设置或接口
